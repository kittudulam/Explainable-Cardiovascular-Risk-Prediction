Explainable Cardiovascular Risk Prediction
Overview

This repository contains the Python implementation accompanying the paper

An Explainable and Clinically Interpretable Machine Learning Framework for Ethical Cardiovascular Risk Prediction

The project develops transparent machine learning models for cardiovascular risk prediction using Logistic Regression, together with explainability, calibration analysis, fairness analysis, SHAP explanations, and external validation.

Datasets

This project uses publicly available datasets. Please download the following datasets from their original sources and place them in the project directory before running the scripts:

Framingham Heart Study Dataset
UCI Heart Disease Dataset
Kaggle Cardiovascular Disease Dataset (cardio_train.csv)
